# Capítulo 7 - Cadeas de texto

{:: encoding="utf-8" /}
{footnotes-off}

I> Un dos grandes cambios que chegaron con ECMA6 son as *[plantillas de cadeas de
I> texto](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/template_strings)*.
I> Que permítennos almacenar un texto con espazos, tabulacións, saltos de liñas... e
I> ademais apórtanos postprocesado e interpolación de expresións.

## Propiedades

- ***[.length()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/length)***

    Devolve a lonxitude da cadea, o que resulta moi importante xa que poderemos utilizar bucles pra recorrer cada unha das letras...

    {lang=js}
    ~~~~~~~
    var cadea = "JavaScript, ¡Inspírate!";
    console.log("JavaScript, ¡Inspírate! ten " + cadea.length + " caracteres.");
    console.log("Unha cadea baleira ten " + ''.length + " caracteres.");
    ~~~~~~~

## Métodos

Facemos unha recopilación simplificada dos métodos máis utilizados, anque existen muchos máis.


- ***[.toLowerCase()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/toLowerCase)***

    Devolve todo en minúsculas.

    {lang=js}
    ~~~~~~~
    console.log('¡INSPÍRATE!'.toLowerCase());
    ~~~~~~~


- ***[.toUpperCase()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/toUpperCase)***

    Devolve todo en maiúsculas.

    {lang=js}
    ~~~~~~~
    console.log('¡inspírate!'.toUpperCase());
    ~~~~~~~

{pagebreak}

- ***[.fromCharCode()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/fromCharCode)***

    Devolve unha cadea creada mediante unha secuencia *[Unicode](https://www.wikiwand.com/es/Unicode)*.

    {lang=js}
    ~~~~~~~
	console.info("É o ano 2016 ("+ String.fromCharCode(8559,8559,8553,8548,8544) + ")");
    ~~~~~~~


- ***[.anchor()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/anchor)***

    Crea un *[link interno (ancla)](https://developer.mozilla.org/es/docs/Web/HTML/Elemento/a)* HTML.

    {lang=js}
    ~~~~~~~
	document.body.innerHTML = "Contidos".anchor("contidos");
	// "<a name="contidos">Contidos</a>"
    ~~~~~~~


- ***[.charAt()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/charAt)***

    Devolve o carácter específico.

    {lang=js}
    ~~~~~~~
    var cadea = "JavaScript, ¡Inspírate!";
    console.log("El carácter(posición) 3 é '" + cadea.charAt(3) + "'")
    ~~~~~~~


- ***[.charCodeAt()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/charCodeAt)***

    Devolve o valor Unicode.

    {lang=js}
    ~~~~~~~
    var cadea = "JavaScript, ¡Inspírate!";
    console.log("O carácter(posición) 3 é '" + cadea.charAt(3) + "', en unicode ("+cadea.charCodeAt(3)+")")
    ~~~~~~~


- ***[.concat()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/concat)***

    Combina o texto de dúas o máis cadeas.

    {lang=js}
    ~~~~~~~
    var cadea1 = "Oh ";
    var cadea2 = "que marabillosa ";
    var cadea3 = "maña'.";
    var combinacion = cadea1.concat(cadea2,cadea3);
    console.log(combinacion); // devolve "Oh que marabillosa maña'."
    ~~~~~~~


- ***[.indexOf()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/indexOf)***

    Devolve a posición do elemento si é capaz de localizalo ou -1 en caso contrario.

    {lang=js}
    ~~~~~~~
    "Mundo Web".indexOf("Web") // 6
    "Mundo Web".indexOf("web") // -1
    ~~~~~~~


- ***[.lastIndexOf()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/lastIndexOf)***

    Devolve a ultima localización do elemento si é capaz de atopalo ou -1 en caso contrario.

    {lang=js}
    ~~~~~~~
    "JavaScript, ¡Inspírate!".lastIndexOf("c"); // 16
    "JavaScript, ¡Inspírate!".lastIndexOf("b"); // -1
    ~~~~~~~

- ***[.link()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/link)***

    Crea un enlace.

    {lang=js}
    ~~~~~~~
    var textoActivo="JavaScript, ¡Inspírate!"
    var url="https://leanpub.com/javascript-inspirate/"
    document.body.innerHTML = "Faga click pra volver a " + textoActivo.link(url);
    ~~~~~~~


- ***[.trim()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/Trim)***

    Elimina espazos baleiros ao principio e ao final da cadea.

    {lang=js}
    ~~~~~~~
    console.log('    ¡inspírate! '.trim()); // ¡Inspírate!
    ~~~~~~~


- ***[.slice()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/slice)***

    Devolve unha cadea nova cunha sección de outra.

    {lang=js}
    ~~~~~~~
    var cadea = "JavaScript, ¡Inspírate!";
    console.log(cadea.slice(12));      // ¡Inspírate!
    console.log(cadea.slice(0, 10));   // JavaScript
    console.log(cadea.slice(13, -1));  // Inspírate
    ~~~~~~~


- ***[.split()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/split)***

    Divide unha cadea usando un separador e retorna un array.

    {lang=js}
    ~~~~~~~
    var cadenaMeses = "Jan,Feb,Mar,Apr,May,Jun,Jul,Aug,Sep,Oct,Nov,Dec";
    var meses = cadenaMeses.split(",");
    console.log("Os meses son un Array?", Array.isArray(meses))
    ~~~~~~~

- ***[.substr()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/substr)***

    Devolve os caracteres dunha cadea, comenzando na localización especificada, é no número de caracteres especificado.

    {lang=js}
    ~~~~~~~
    var cadea = "JavaScript, ¡Inspírate!";
    console.log("cadea.substr(12, 12):", cadea.substr(12, 12));
    console.log("cadea.substr(0,10):", cadea.substr(0, 10));
    ~~~~~~~


- ***[.substring()](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/String/substring)***


    Devolve un subconxunto.

    {lang=js}
    ~~~~~~~
    var cadea = "JavaScript, ¡Inspírate!";
    console.log("(0,5): "    + cadea.substring(0,10));
    ~~~~~~~
